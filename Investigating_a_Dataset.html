# These functions remove rows with missing values in the specified columns
# This ensures that the analysis is based on complete data
def remove_missing_values(df, columns):
    for column in columns:
        df = df[df[column].isnull() == False]
    return df

def remove_zero_values(df, columns):
    for column in columns:
        df = df[df[column] != 0]
    return df

# Read the CSV file into a Pandas DataFrame
df = pd.read_csv('tmdb-movies.csv')

# Clean the data
missing_value_columns = ["cast", "genres", "homepage", "director", "tagline", "keywords", "production_companies"]
df = remove_missing_values(df, missing_value_columns)

zero_value_columns = ["budget_adj", "revenue_adj"]
df = remove_zero_values(df, zero_value_columns)

# Print the number of missing values in each column
# This helps identify any remaining missing data

Question 1: In which month were the most movies released?

In[34]:
# Use this, and more code cells, to explore your data. Don't forget to add

# Convert the 'release_date' column to datetime objects
df['release_date'] = pd.to_datetime(df['release_date'])

# Count the number of movie releases for each month
number_of_release = df['release_date'].dt.month.value_counts().sort_index()
# Create a list of month names
months = ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec']
# Convert the Series to a DataFrame and reset the index
number_of_release = pd.DataFrame(number_of_release).reset_index()
# Rename the columns
number_of_release.columns = ['month_number', 'number_of_release']
# Add a column for the month names
number_of_release['month'] = months

# Create a line plot of the number of movie releases per month
number_of_release.plot(x='month', y='number_of_release', kind='line', marker='o', fontsize=11, figsize=(8, 6))
# Set the plot title
mpl.title('Movie Released per Month', fontsize=15)
# Set the x-axis label
mpl.xlabel('Month', fontsize=13)
# Set the y-axis label
mpl.ylabel('Number of Movie Releases', fontsize=13)
# Display the plot
mpl.xticks(rotation=45)
mpl.grid()
mpl.show()

From this line chart, we can see that September has the most releases when compared with all the months.

Question 2: Which actors have the most appearances in movies?

In[26]:
# Continue to explore the data to address your additional research

# Initialize a dictionary to count actor appearances
actor_dict = {}
actors = df["cast"]
actors = actors.str.split("|")
actors = np.array(actors)

# Count appearances of each actor
for actorList in actors:
    for actor in actorList:
        actor = actor.lstrip()
        if actor not in actor_dict:
            actor_dict[actor] = 1
        else:
            actor_dict[actor] += 1

# Sort the actor dictionary by number of appearances
sorted_actor_dict = sorted(actor_dict.items(), key = operator.itemgetter(1), reverse = True)

# Prepare data for visualization
x_axis = list()
y_axis = list()
for item in sorted_actor_dict[0:10]:
    x_axis.append(item[0])
    y_axis.append(item[1])


# Set up the visualization
sb.set(rc={'figure.figsize': (12, 10)}, font_scale=1.4)
x = sb.barplot(x=x_axis, y=y_axis, palette="Set3")

# Rotate x-axis labels for better readability
for item in x.get_xticklabels():
    item.set_rotation(85)

#Set labels and title
x.set(xlabel='actor names', ylabel='Number of Appearances', title='Top 10 Actors with Most Appearances in Movies')
# Show the plot
mpl.show()

We can see that the top ten actors include A-list celebrities like Samuel L. Jackson, Brad Pitt, Matt Damon, and Harrison Ford. It makes sense that these household names would be found in a large number of movies.

Conclusions

Question 1 - A statistical inference we can make is that films are more likely to be released at the end of the year as we can see on the graph that most films were released near the end of the year with the months of September, October, November,and Decemeber having more releases on average. A limitation that exists relates to the data accuracy. There are only around 10,000 films found in this csv file. However, a quick Google search shows that 100,000 films have been created, showing that while the question is answered for the sample found in the CSV list, it does not accurately represent all films ever made.

However, similar to question 1, a limitation that exists relates to the data accuracy. There are only around 10,000 films found in this csv file. However, a quick Google search shows that 100,000 films have been created, showing that while the question is answered for the sample found in the CSV list, it does not accurately represent all films ever made.
